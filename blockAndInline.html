Elementos block

Um elemento em nível de bloco sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço (uma margem) antes e depois do elemento.

Elementos block mais utilizados são o p e div

Elementos Inline

Um elemento embutido não começa em uma nova linha. Um elemento embutido ocupa apenas a largura necessária.

Inline mais usado é o span

O elemento div

  • O div elemento é frequentemente usado como contêiner para outros elementos HTML.
  • O div elemento não possui atributos obrigatórios, mas style, classe idsão comuns.
  • O div com css pode ser usado para estilizar blocos de conteudo.
