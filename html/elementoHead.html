Elemento head

Title

  • define um título na barra de ferramentas do navegador
  • fornece um título para a página quando ela é adicionada aos favoritos
  • exibe um título para a página nos resultados do mecanismo de pesquisa

Elemento style

  • O style é usado para definir informações de estilo para uma única página html

Elemento link

  • O link define o relacionamento entre o documento atual e um recurso externo.

Elemento meta

normalmente é usado para especificar o conjunto de caracteres, a descrição da página, as palavras-chave, o autor do documento e as configurações da janela de visualização.

Elemento script

O elemento é usado para definir JavaScripts do lado do cliente.

Elemento base

especifica o URL base e/ou destino para todos os URLs relativos em uma página
